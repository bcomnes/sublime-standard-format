Standard Format
---------------
This auto-formatter requires that you install `standard` as a devDependency in order to function properly:

  $ npm i standard --save-dev

Please file any issues with the plug-in you come across at:

  https://github.com/bcomnes/sublime-standard-format/issues

Auto-formatting issues can be directed at:

  https://github.com/feross/standard/issues

Please view the README for more information/documentation.

Consider installing the `standard` linter plug-in:

  https://github.com/Flet/SublimeLinter-contrib-standard
